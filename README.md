# Paradise Nursery - E-Plant Shopping Application

Paradise Nursery is a modern, responsive web application built with React and Redux Toolkit, designed for plant enthusiasts to explore and purchase a wide variety of plants. From air-purifying indoor plants to aromatic garden additions, we bring nature closer to you.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nileshcodehub/e-plantShopping.git
   cd e-plantShopping
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
